Как найти СДЭК ID в приложении: подробная инструкция с картинками

Идентификатор СДЭК ID — это уникальный номер вашего аккаунта в системе транспортной компании, который нужен для отслеживания посылок, оплаты услуг и работы с API. Без него невозможно воспользоваться многими функциями личного кабинета, особенно если вы бизнес-клиент или интегрируете сервис с другими платформами. Однако не все пользователи знают, где именно искать этот номер в мобильном приложении СДЭК — особенно после обновлений интерфейса 2023–2026 годов.

В этой статье мы разберём все актуальные способы найти СДЭК ID в приложениях для Android и iOS, а также расскажем, что делать, если идентификатор не отображается или отличается от ожидаемого. Отдельно остановимся на типичных ошибках и нюансах для бизнес-аккаунтов.

Что такое СДЭК ID и зачем он нужен

СДЭК ID — это 10-значный числовой код, который присваивается каждому зарегистрированному пользователю (физическому или юридическому лицу). Он используется для:

  • 📦 Отслеживания посылок через API (например, для автоматической синхронизации с интернет-магазином).
  • 💳 Оплаты услуг СДЭК через банковские сервисы или платежные системы.
  • 🔄 Интеграции с CRM-системами (1С, Bitrix24, MoySklad).
  • 📋 Подтверждения личности при обращении в поддержку.

Важно не путать СДЭК ID с номером заказа или трек-номером посылки. Первый — это ваш личный идентификатор в системе, а вторые относятся к конкретным отправлениям. Например, трек-номер выглядит как 123456789, а СДЭК ID — как 1234567890 (10 цифр).

⚠️ Внимание: Если вы видите в приложении номер короче 10 символов (например, 6–8 цифр), это не СДЭК ID, а внутренний код клиента. Для API и оплаты он не подходит!

Где искать СДЭК ID в мобильном приложении (Android/iOS)

Интерфейс приложения СДЭК периодически обновляется, но путь к идентификатору остаётся стабильным. Вот пошаговая инструкция для последних версий (2026 год):

  1. Откройте приложение и авторизуйтесь (если не вошли автоматически).
  2. Тапните по иконке профиля в правом нижнем углу (или в верхнем меню на iOS).
  3. Выберите раздел Личные данные или Профиль.
  4. Прокрутите вниз до блока Информация о клиенте — там будет строка СДЭК ID.

На некоторых версиях приложения идентификатор может скрываться за кнопкой Показать реквизиты или Данные для API. Если вы не видите номер сразу, проверьте эти вкладки.

Отличия для физических и юридических лиц

Если вы зарегистрированы как физическое лицо, СДЭК ID обычно отображается сразу в профиле. Для юридических лиц и ИП процесс может отличаться:

Тип аккаунта Где искать СДЭК ID Особенности
Физическое лицо Профиль → Личные данные Обычно 10 цифр, начинается с 1 или 2
Юридическое лицо / ИП Профиль → Реквизиты компании → Данные для API Может быть привязан к договору, начинается с 3–5
Корпоративный клиент Требуется запрос в поддержку ID выдаётся отдельно для каждого филиала

Для бизнес-аккаунтов СДЭК ID иногда дублируется в личном кабинете на сайте (раздел Мои данные → Реквизиты). Если в приложении номер отсутствует, проверьте веб-версию.

⚠️ Внимание: У юридических лиц может быть несколько СДЭК ID — например, отдельные идентификаторы для разных точек выдачи или типов отправлений. Уточняйте актуальный номер в договоре или у менеджера.

Что делать, если СДЭК ID не отображается

Если вы выполнили все шаги, но идентификатор не виден, причины могут быть следующими:

  • 🔄 Устаревшая версия приложения. Обновите его через App Store или Google Play.
  • 📱 Неполная синхронизация данных. Попробуйте выйти из аккаунта и зайти заново.
  • 🔒 Ограничения по типу аккаунта. Например, у некоторых тарифов для физических лиц ID не выдаётся.
  • 🛠️ Технические работы. Проверьте статус сервисов СДЭК на сайте cdek.ru.

Если проблема не решается, свяжитесь с поддержкой через:

  • 💬 Чат в приложении (раздел Помощь).
  • ☎️ Телефон горячей линии: 8 800 250-04-05.
  • ✉️ Электронную почту: support@cdek.ru.
Как ускорить ответ от поддержки?

При обращении сразу укажите: 1) тип аккаунта (физ./юр. лицо), 2) привязанный email/телефон, 3) скриншот раздела профиля без СДЭК ID. Это сократит время обработки запроса с 24 до 2–4 часов.

Как найти СДЭК ID в веб-версии личного кабинета

Если мобильное приложение не показывает идентификатор, попробуйте получить его через веб-версию:

  1. Перейдите на сайт lk.cdek.ru и авторизуйтесь.
  2. В верхнем меню выберите Мои данные.
  3. Прокрутите до блока Реквизиты для оплаты или Информация о клиенте.
  4. СДЭК ID будет указан рядом с наименованием компании (для юрлиц) или ФИО (для физлиц).

В некоторых случаях номер может скрываться за кнопкой Показать реквизиты или Экспорт данных. Если вы не видите ID, попробуйте:

  • 🔍 Использовать функцию поиска по странице (Ctrl+F) и ввести "СДЭК ID".
  • 📄 Скачать выписку по аккаунту — иногда номер указывается в PDF-документе.
💡

Если вы часто работаете с API СДЭК, сохраните свой ID в защищённом месте (например, в менеджере паролей). При смене тарифа или реквизитов номер может измениться!

Типичные ошибки при поиске СДЭК ID

Пользователи часто путают СДЭК ID с другими номерами. Вот самые распространённые ошибки:

Что путают с СДЭК ID Как отличить
Номер заказа Содержит буквы (например, CX123456789RU), а СДЭК ID — только цифры.
Трек-номер посылки Начинается с букв (RR, CX) и короче (9 символов).
Номер договора Указан в документах, но не подходит для API.
Логин от личного кабинета Может быть email или телефон, а СДЭК ID — только цифры.

Ещё одна частая ошибка — попытка использовать идентификатор точки выдачи (например, MOW123) вместо личного СДЭК ID. Эти номера не взаимозаменяемы!

💡

СДЭК ID — это всегда 10 цифр без букв и символов. Если вы видите другой формат, это не тот номер.

Как использовать СДЭК ID для работы с API

Если вам нужен идентификатор для интеграции с 1С, WooCommerce или другими системами, недостаточно просто скопировать номер. Потребуются дополнительные данные:

  • 🔑 API-ключ (генерируется в личном кабинете в разделе Настройки → API).
  • 🌐 Адрес сервера (например, https://api.cdek.ru/v2).
  • 📝 Тип аккаунта (тестовый или боевой).

Пример запроса для получения информации о заказе через API:

curl -X POST 'https://api.cdek.ru/v2/orders'

-H 'Authorization: Bearer {Ваш_API_ключ}'


-H 'Content-Type: application/json'


-d '{


"type": 1,


"number": "CX123456789RU",


"cdek_id": "1234567890"  // Здесь подставляете ваш СДЭК ID


}'

Без корректного СДЭК ID сервер вернёт ошибку 403 Forbidden или 401 Unauthorized. Убедитесь, что вы используете актуальный номер — при смене тарифа или реквизитов он может обновляться.

FAQ: Частые вопросы о СДЭК ID

Можно ли узнать СДЭК ID без авторизации?

Нет, идентификатор привязан к аккаунту и отображается только после входа в систему. Если вы забыли пароль, восстановите доступ через email или телефон.

Что делать, если СДЭК ID изменился?

Это возможно при смене тарифа, реквизитов или типа аккаунта. Обновите номер в всех интеграциях (CRM, платежные системы) и проверьте работу API. Старый ID перестанет действовать через 30 дней.

Нужно ли платить за получение СДЭК ID?

Нет, идентификатор присваивается автоматически при регистрации и бесплатен. Оплачиваются только дополнительные услуги (API-запросы сверх лимита, премиум-тарифы).

Можно ли использовать один СДЭК ID для нескольких аккаунтов?

Нет, каждый аккаунт (даже если привязан к одному человеку) имеет уникальный идентификатор. Для бизнес-клиентов с несколькими точками выдачи выдаются отдельные ID.

Как проверить, что СДЭК ID действующий?

Сделайте тестовый запрос через API (например, проверку статуса несуществующего заказа). Если сервер вернёт ошибку аутентификации, ID неактивен.
