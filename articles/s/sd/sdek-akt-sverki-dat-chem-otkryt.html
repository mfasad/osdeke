Файл акта сверки СДЭК в формате DAT: чем открыть и как правильно использовать

Если вы работаете с транспортной компанией СДЭК как юридическое лицо или ИП, то регулярно сталкиваетесь с необходимостью сверки взаимных расчётов. Компания предоставляет акты сверки в формате DAT — бинарном виде, который не открывается стандартными текстовыми редакторами. Многие клиенты теряются: как просмотреть содержимое такого файла, проверить корректность данных или конвертировать его в удобный формат?

Проблема усугубляется тем, что СДЭК не всегда прилагает инструкции по работе с DAT-файлами, особенно если речь идёт о массовой выгрузке документов через личный кабинет. В этой статье мы разберём все способы открытия акта сверки, анализируем структуру файла и даём рекомендации по автоматизации обработки. Вы узнаете, какие программы и онлайн-сервисы подходят для этой задачи, а также как избежать типичных ошибок при работе с бинарными данными.

Важно: формат DAT от СДЭК — это не просто текстовый файл с другим расширением. Он содержит структурированные данные в проприетарном виде, которые требуют специального ПО для декодирования. Попытки открыть его через Блокнот или Excel приведут к нечитаемому набору символов.

Почему СДЭК использует формат DAT для актов сверки

Формат DAT (от англ. data — данные) выбран компанией не случайно. Он позволяет:

  • 🔒 Защитить данные от случайного изменения — в отличие от Excel или CSV, бинарный файл сложнее подделать.
  • 📦 Сократить размер — DAT-файлы компактнее текстовых аналогов, что важно при массовой отправке документов.
  • 🤖 Автоматизировать обработку — структурированные данные легче интегрировать в бухгалтерские программы (1С, МойСклад и др.).
  • 🔄 Обеспечить совместимость с внутренними системами СДЭК, где DAT — стандарт для обмена данными.

Однако для конечного пользователя это создаёт сложности. Если вы получаете акт сверки впервые, скорее всего, у вас нет подходящего софта для его просмотра. К счастью, решений несколько — от официальных утилит СДЭК до универсальных программ для работы с бинарными файлами.

⚠️ Внимание: Некоторые DAT-файлы от СДЭК могут быть зашифрованы или подписаны электронной подписью. В этом случае для открытия потребуется сертификат ключа проверки, который компания присылает отдельно по запросу.

Официальные способы открытия DAT-файлов от СДЭК

Самый надёжный метод — использовать инструменты, рекомендованные самой компанией. СДЭК предоставляет два варианта:

1. Личный кабинет на сайте СДЭК

Если акт сверки пришёл в виде вложения к письму или доступен в разделе Документы → Акт сверки, попробуйте следующее:

  1. Авторизуйтесь на сайте lk.cdek.ru.
  2. Перейдите в раздел Финансы → Акт сверки.
  3. Найдите нужный документ и нажмите Просмотреть — система автоматически декодирует DAT и покажет данные в удобном виде.

🔹 Плюсы: не требует установки ПО, данные отображаются в структурированном виде с возможностью экспорта в PDF или Excel.

🔸 Минусы: работает только для актуальных актов (старые документы могут быть архивированы и недоступны для просмотра онлайн).

2. Программа СДЭК-Коннект

Для клиентов с большим документооборотом компания разработала десктопное приложение СДЭК-Коннект. Оно позволяет:

  • 📂 Открывать DAT-файлы напрямую с жёсткого диска.
  • 🔍 Просматривать детализацию по каждой накладной.
  • 📤 Экспортировать данные в Excel или 1С.

📌 Как установить:

  1. Скачайте программу с официального сайта.
  2. Установите и авторизуйтесь с данными от личного кабинета.
  3. Перетащите DAT-файл в окно программы или выберите его через меню Файл → Открыть.

⚠️ Внимание: СДЭК-Коннект требует регулярного обновления. Если файл не открывается, проверьте актуальность версии программы в разделе Справка → О программе.

💡

Если в СДЭК-Коннект акт сверки отображается некорректно, попробуйте очистить кэш программы через меню Настройки → Очистка данных.

Как выглядит акт сверки в HEX-редакторе?

В HxD DAT-файл от СДЭК отображается как набор байтов, где первые 16–32 символа обычно содержат заголовок (например, CDK_ACT_2026), далее идут блоки с данными о накладных, суммах и реквизитах. Без документации от СДЭК расшифровать их вручную крайне сложно.

Структура DAT-файла акта сверки: что внутри

Файлы акта сверки от СДЭК имеют уникальную бинарную структуру, которая включает следующие обязательные блоки:

  1. Заголовок — содержит версию формата, дату создания акта и идентификатор клиента.
  2. Список накладных — перечень всех отправлений с номерами, датами и статусами.
  3. Финансовые данные — суммы к оплате, удержанные комиссии, НДС.
  4. Реквизиты сторон — данные СДЭК и вашей компании (ИНН, КПП, адреса).
  5. Электронная подпись (опционально) — если файл подписан, блок содержит сертификат и хеш.

Пример начала файла в HEX-формате: 

43 44 4B 5F 41 43 54 5F 32 30 32 34 00 00 00 01  CDK_ACT_2026....

1A 03 00 00 4E 41 4B 4C 31 32 33 34 35 36 00 00  ....NAKL123456..

🔹 Как это помогает? Зная структуру, можно вручную извлечь данные через HEX-редактор или написать скрипт для автоматической расшифровки (например, на Python).

⚠️ Внимание: Если в файле присутствует блок электронной подписи (обычно в конце), его удаление или изменение сделает документ недействительным. Для проверки подписи используйте программу КриптоПро CSP.

Конвертация DAT в Excel: пошаговая инструкция

Если вам нужно перенести данные акта сверки в Excel для дальнейшей обработки, следуйте этому алгоритму:

Способ 1: Через СДЭК-Коннект

  1. Откройте файл в программе СДЭК-Коннект.
  2. Нажмите Файл → Экспортировать → Excel (.xlsx).
  3. Выберите папку для сохранения и подтвердите экспорт.

Способ 2: Через HEX-редактор + ручной перенос

Если СДЭК-Коннект недоступен:

  1. Откройте DAT-файл в HxD Hex Editor.
  2. Найдите блоки с данными (ищите текстовые фрагменты вроде NAKL, SUM).
  3. Скопируйте их в Блокнот, затем импортируйте в Excel через Данные → Из текста.

📌 Совет: Для автоматической конвертации можно использовать скрипт на Python с библиотекой struct. Пример кода: 

import struct


with open('akt.dat', 'rb') as f:


header = f.read(16)  # Читаем заголовок


print(header.decode('utf-8'))  # Выводим название акта

⚠️ Внимание: При ручной конвертации высок риск ошибок в финансовых данных. Всегда сверяйте итоговые суммы с оригинальным актом!

Типичные ошибки и их решения

При работе с DAT-файлами пользователи сталкиваются с следующими проблемами:

  • 🚫 "Файл повреждён" — чаще всего возникает при неполной загрузке. Решение: запросите акт повторно через личный кабинет.
  • 🔑 "Неверная подпись" — означает, что файл был изменён или сертификат устарел. Решение: скачайте актуальный сертификат в разделе Настройки → Сертификаты.
  • 📉 "Некорректные суммы" — может быть вызвано ошибкой экспорта. Решение: сравните данные с PDF-версией акта (если прилагается).
  • 🖥️ "Не открывается в Excel" — DAT не является табличным форматом. Решение: используйте СДЭК-Коннект или конвертируйте через HEX-редактор.

🔹 Если ничего не помогает: обратитесь в службу поддержки СДЭК с указанием:

  • Номера акта сверки.
  • Скриншота ошибки.
  • Версии используемого ПО.

📞 Контакты поддержки:

  • Телефон: 8 800 250-04-05 (бесплатно по РФ).
  • Email: support@cdek.ru.
  • Чат в личном кабинете (раздел Помощь).

Автоматизация обработки актов сверки

Если ваша компания ежемесячно получает десятки актов, ручная обработка становится неэффективной. Рассмотрите следующие варианты автоматизации:

1. Интеграция с 1С

Многие бухгалтерские конфигурации 1С поддерживают импорт DAT-файлов от СДЭК. Для этого:

  1. Установите обработку ОбменСДЭК.epf (доступна у партнёров 1С).
  2. Настройте правила сопоставления полей (номер накладной → номер документа в 1С).
  3. Запустите загрузку через Файл → Открыть.

2. API СДЭК

Для IT-специалистов: компания предоставляет API, через который можно получать акты сверки в формате JSON или XML. Пример запроса: 

GET https://api.cdek.ru/v2/finance/acts?date_from=2026-01-01

Headers:


Authorization: Bearer {ваш_токен}

3. Собственные скрипты

Если у вас есть программист, можно разработать парсер DAT-файлов на Python или C#. Библиотеки для работы с бинарными данными:

  • Python: struct, pycryptodome (для подписи).
  • C#: BinaryReader.

🔹 Пример задачи для автоматизации: еженедельная выгрузка актов в Google Sheets с расчётом дебетовой/кредитовой задолженности.

FAQ: Частые вопросы по работе с DAT-файлами СДЭК

Можно ли открыть DAT-файл без специальных программ?

Технически да — через HEX-редактор, но без знания структуры файла данные будут нечитаемыми. Для простого просмотра используйте СДЭК-Коннект.

Почему в акте сверки не совпадают суммы с моими записями?

Причины могут быть разные:

  • 📅 У вас учтены не все накладные за период.
  • 💰 СДЭК удержал комиссию за дополнительные услуги (страховка, упаковка).
  • ⚖️ Ошибка в ручном учёте — сверьте данные с оригиналами накладных.

Запросите у менеджера СДЭК детализацию по спорным позициям.

Как проверить электронную подпись в DAT-файле?

Используйте программу КриптоПро CSP:

  1. Установите сертификат СДЭК (прилагается к акту или скачивается в личном кабинете).
  2. Откройте файл через Просмотр → Подписи.
  3. Нажмите Проверить — программа покажет статус подписи.

Можно ли редактировать DAT-файл?

🚫 Нет, любые изменения сделают документ недействительным. Если нужно внести правки (например, исправить реквизиты), запросите новый акт у СДЭК.

Где хранить архив DAT-файлов?

Рекомендации:

  • 📁 Создайте отдельную папку на жёстком диске или в облаке (Google Drive, Яндекс.Диск).
  • 📌 Назовите файлы по шаблону: Акт_СДЭК_2026-05_DAT.
  • 🔒 Сделайте резервную копию на внешнем носителе (на случай сбоя ПК).
