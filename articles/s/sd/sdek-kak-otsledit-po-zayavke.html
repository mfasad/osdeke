Почему отслеживание по номеру заявки СДЭК — самый надёжный способ?

Номер заявки в СДЭК — это уникальный идентификатор, который присваивается грузу ещё на этапе оформления заказа. В отличие от трек-номера, который может дублироваться для разных отправлений (например, при массовой рассылке), номер заявки привязан конкретно к вашему договору и заказу. Это делает его незаменимым инструментом, если вы отправляете или получаете грузы регулярно, особенно для бизнес-клиентов.

В 2026 году СДЭК расширил функционал отслеживания по номеру заявки: теперь можно не только видеть текущий статус, но и получать расширенную информацию о маршруте, предполагаемой дате доставки и даже истории изменений (например, если адрес получателя корректировался). Однако многие пользователи до сих пор не знают, как правильно использовать этот номер — особенно когда трек-номер ещё не сформирован или потерян.

В этой статье разберём все актуальные способы отслеживания по номеру заявки, включая скрытые возможности личного кабинета, API для автоматизации и альтернативные сервисы. А также расскажем, что делать, если система не находит вашу заявку — это частая проблема при работе с корпоративными аккаунтами или при отправке через партнёров СДЭК.

Чем номер заявки отличается от трек-номера?

Многие путают эти два идентификатора, но они служат разным целям:

  • 📌 Номер заявки — внутренний идентификатор заказа в системе СДЭК, формируется сразу при оформлении. Пример: 123456789/2026-05-15. Используется для управления грузом до момента передачи курьеру.
  • 🚚 Трек-номер — внешний номер для отслеживания перемещения посылки. Пример: 111122223333. Появляется только после того, как груз принят на склад СДЭК.

Ключевое отличие: номер заявки позволяет отслеживать статус до того, как посылка физически попадёт в логистическую сеть (например, если вы оформили заказ, но ещё не передали его курьеру). Трек-номер же работает только после сканирования груза на складе.

Для бизнес-клиентов номер заявки также важен для:

  • 📊 Сверки с бухгалтерией (привязка к счёту или акту выполненных работ).
  • 🔄 Массового управления заказами (например, отмена или редактирование нескольких отправлений одновременно).
  • 📑 Получения отчётности по всем заявкам за период.

Способ 1: Отслеживание через личный кабинет СДЭК

Самый надёжный и информативный метод — использовать личный кабинет СДЭК. Здесь доступны все данные по заявке, включая историю изменений и документы. Инструкция:

  1. Авторизуйтесь в личном кабинете (для физических лиц — через email/телефон, для юридических — по договору).
  2. Перейдите в раздел Мои заказы → Заявки на доставку.
  3. Введите номер заявки в поле поиска (формат: XXXXXXXX/ГГГГ-ММ-ДД).
  4. Нажмите на строку с нужной заявкой, чтобы открыть детальную информацию.

В карточке заявки вы увидите:

  • 📦 Статус обработки (например, "Ожидает передачи курьеру" или "На складе отправителя").
  • 📅 Планируемую дату доставки (если груз уже в пути).
  • 💰 Стоимость услуг с разбивкой по тарифам.
  • 📄 Прикреплённые документы (накладные, акты, чеки).

Способ 2: Отслеживание по API СДЭК (для автоматизации)

Если вы отправляете грузы регулярно (например, как интернет-магазин), ручное отслеживание каждой заявки неэффективно. В этом случае поможет API СДЭК — инструмент для интеграции отслеживания в вашу CRM или сайт.

Чтобы получить данные по заявке через API:

  1. Зарегистрируйтесь как разработчик на портале API СДЭК и получите ключ доступа.
  2. Используйте эндпоинт /orders с параметром order_uuid (уникальный идентификатор заявки). Пример запроса:
GET https://api.cdek.ru/v2/orders?order_uuid=your_order_uuid

Headers:


Authorization: Bearer ваш_токен


Content-Type: application/json

В ответе вы получите JSON с полной информацией о заявке, включая:

  • 📌 status — текущий статус (например, "accepted" или "in_transit").
  • 📦 packages — данные о грузах (вес, габариты).
  • 📍 route — маршрут с указанием складов и сроков.

Для массового отслеживания можно отправлять пакетные запросы с несколькими order_uuid.

Пример ответа API для заявки в статусе "На складе"

⚠️ Внимание: При работе с API учитывайте лимиты запросов (бесплатный тариф — до 1000 запросов/день). Для большого потока заявок оформляйте платный доступ.

Способ 3: Мобильное приложение СДЭК

Официальное приложение СДЭК (доступно для iOS и Android) позволяет отслеживать заявки прямо со смартфона. Преимущества:

  • 🔔 Уведомления о изменении статуса (например, "Груз прибыл на склад").
  • 📱 Быстрый доступ через сканирование QR-кода на накладной.
  • 🗺️ Карта с отображением текущего местоположения груза (если доступно).

Как найти заявку в приложении:

  1. Откройте раздел Отслеживание.
  2. Выберите вкладку Заявки (не "Посылки"!).
  3. Введите номер в формате XXXXXXXX/ГГГГ-ММ-ДД.

Если заявка не отображается, обновите данные (кнопка 🔄 в правом верхнем углу) или проверьте подключение к интернету. В приложении также можно:

  • 📞 Позвонить в поддержку прямо из карточки заявки.
  • 📎 Прикрепить фотографии груза (полезно при претензиях).
  • 📅 Перенести дату доставки (если груз ещё не в пути).
💡

В настройках приложения включите опцию "Синхронизация с календарём" — тогда даты доставки будут автоматически добавляться в ваш Google Календарь или Apple Calendar.

Способ 4: Отслеживание через партнёров СДЭК

Если вы оформили заявку через посредника (например, маркетплейс Ozon, Wildberries или агрегатор доставки), номер заявки СДЭК может отличаться от внутреннего номера заказа магазина. В этом случае:

  1. Найдите в личном кабинете партнёра раздел "Логистика" или "Доставка".
  2. Скачайте накладную (обычно в формате PDF) — там будет указан номер заявки СДЭК.
  3. Используйте этот номер для отслеживания на сайте СДЭК.

Примеры, где это актуально:

Партнёр Где искать номер заявки СДЭК Пример формата
Ozon Раздел "Мои заказы" → "Детали доставки" → "Накладная СДЭК" 87654321/2026-04-10
Wildberries "Логистика" → "Акты передачи" (в Excel-файле) WB-12345678 (требует конвертации)
Яндекс Маркет "Заказы" → "Информация о доставке" → "Трек-номер" 11223344/2026-03-15

Если номер заявки не отображается, свяжитесь с поддержкой партнёра и запросите накладную СДЭК — без неё отслеживание через стандартные каналы невозможно.

⚠️ Внимание: Некоторые агрегаторы (например, Boxberry или PickPoint) используют собственные трек-номера, которые не совместимы с системой СДЭК. Уточняйте у отправителя, через какую именно компанию идёт груз.

Способ 5: Альтернативные сервисы отслеживания

Если по какой-то причине вы не можете использовать официальные каналы СДЭК, существуют сторонние сервисы для отслеживания заявок. Они полезны, когда:

  • 🌍 Нужно отслеживать грузы из-за границы (например, из Китая через СДЭК Global).
  • 📊 Требуется сводная аналитика по нескольким перевозчикам.
  • 🔍 Номер заявки не находится в личном кабинете (иногда сторонние сервисы имеют более полные базы).

Популярные платформы:

  • 🔗 17Track — поддерживает СДЭК и 1000+ других перевозчиков. Введите номер заявки в формате CDEKXXXXXXXX.
  • 🔗 Parcels — уведомления о статусах на email, интеграция с Telegram.
  • 🔗 ГдеПосылка — русскоязычный сервис с историей изменений.

Ограничения сторонних сервисов:

  • ❌ Данные могут обновляться с задержкой (до 24 часов).
  • ❌ Не все сервисы корректно обрабатывают номера заявок (чаще работают с трек-номерами).
  • ❌ Риск утечки данных при использовании неофициальных платформ.
💡

Официальный сайт и приложение СДЭК всегда показывают самую актуальную информацию. Сторонние сервисы удобны для массового отслеживания, но не гарантируют 100% точность.

Что делать, если заявка не находится?

Ситуации, когда система не находит номер заявки, встречаются часто. Рассмотрим типичные причины и решения:

Проблема Возможная причина Решение
Номер введён правильно, но система пишет "Не найдено" Заявка ещё не обработана (статус "Черновик") или архивирована Проверьте статус в личном кабинете или обратитесь в поддержку с указанием даты создания
Номер заявки отличается от того, что в накладной Ошибка при ручном вводе или сбой при генерации номера Сверьте номер с оригиналом накладной (PDF или бумажный вариант)
Заявка видна, но статус не обновляется более 3 дней Груз застрял на складе или потерян Напишите в поддержку с просьбой проверить физическое местоположение груза

Если ни один из способов не помог, воспользуйтесь формой обратной связи на сайте СДЭК:

  1. Перейдите на страницу поддержки.
  2. Выберите тему "Поиск груза" → "Не могу найти заявку".
  3. Укажите:
    • Номер заявки (или трек-номер, если есть).
    • Дата оформления.
    • Данные отправителя (ФИО/название компании).
  • Прикрепите скан накладной (если есть).

    • Срок ответа — до 24 часов в будние дни. Для ускорения процесса можно позвонить на горячую линию: 8 800 250-04-05 (звонок бесплатный по России).

    FAQ: Частые вопросы по отслеживанию заявок СДЭК

    Можно ли отследить заявку без номера, только по ФИО?

    Нет, для отслеживания обязательно знать номер заявки или трек-номер. Однако если вы — отправитель, можно найти заявку в личном кабинете по дате создания или сумме заказа. Для получателей такой возможности нет из-за политики конфиденциальности.

    Сколько времени хранится информация о заявке в системе?

    Данные о заявках хранятся 6 месяцев с даты создания. После этого они архивируются, но могут быть восстановлены по запросу в поддержку (потребуется указать дату и данные отправителя).

    Почему в мобильном приложении и на сайте разные статусы?

    Это может происходить из-за кэширования данных в приложении. Обновите информацию вручную (кнопка 🔄) или очистите кэш приложения в настройках смартфона. Если разница сохраняется, обратитесь в поддержку — возможен сбой синхронизации.

    Можно ли отменить заявку, если она уже в статусе "В пути"?

    Отмена возможна, но только до момента передачи груза на склад СДЭК. Если статус "В пути", вы можете:

    • Изменить адрес доставки (платно, через поддержку).
    • Отказаться от груза при получении (без оплаты доставки).

    Полную отмену в этом случае оформить нельзя.

    Как отследить массово несколько заявок?

    Для массового отслеживания используйте:

    • Личный кабинет СДЭК (раздел "Экспорт" → выгрузите данные в Excel).
    • API СДЭК (эндпоинт /orders с параметром order_uuids).
    • Сторонние сервисы вроде ParcelsApp (поддерживают загрузку списка номеров).

    В Excel можно настроить автоматическое обновление статусов через Power Query.
