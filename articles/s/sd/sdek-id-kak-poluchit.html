Что такое СДЭК ID и зачем он нужен

Идентификатор СДЭК ID — это уникальный цифровой код, который присваивается каждому клиенту транспортной компании при регистрации в личном кабинете. Он служит ключом для доступа к расширенным функциям сервиса: от отслеживания посылок до интеграции с Ozon, Wildberries или собственной CRM-системой через API СДЭК.

Без этого идентификатора вы не сможете:

  • 📦 Автоматизировать создание и печать наклеек для заказов
  • 🔄 Отслеживать статус доставки в реальном времени через сторонние платформы
  • 💰 Получать персонализированные тарифы для бизнес-клиентов
  • 📊 Формировать отчёты по отправкам в 1 клик

Физическим лицам СДЭК ID присваивается автоматически при первой отправке посылки через сайт или мобильное приложение. А вот для юридических лиц и интернет-магазинов его получение требует дополнительных шагов — об этом подробнее в следующих разделах.

Кто может получить СДЭК ID: категории пользователей

Контроль за выдачей идентификаторов у СДЭК строгий: не каждый клиент получит доступ к полному функционалу. Разберём, кто имеет право на получение СДЭК ID и какие ограничения действуют в 2026 году.

Категория пользователя Тип СДЭК ID Доступные функции Требуемые документы
Физические лица Личный Отслеживание своих посылок, оплата онлайн Паспортные данные (при первой отправке)
Индивидуальные предприниматели Бизнес Массовая печать наклеек, API, скидки на тарифы ИНН, выписка из ЕГРИП, договор с СДЭК

Партнёры-маркетплейсы Специальный Автоматическая синхронизация заказов Договор о сотрудничестве с платформой

⚠️ Внимание: С 1 марта 2026 года СДЭК ужесточил требования к верификации бизнес-аккаунтов. Теперь для получения корпоративного СДЭК ID обязательно наличие действующего договора с компанией. Без него доступ будет ограничен функционалом для физических лиц.

Если вы планируете использовать идентификатор для интеграции с API, убедитесь, что ваш тарифный план поддерживает этот функционал. Бесплатные аккаунты имеют ограничение в 1000 запросов/месяц.

Пошаговая инструкция: как получить СДЭК ID для физических лиц

Процесс получения идентификатора для обычных клиентов максимально упрощён. Вам не потребуется собирать документы или заключать договоры — достаточно выполнить несколько действий в личном кабинете.

☑️ Как получить СДЭК ID для физического лица

Выполнено: 0 / 4

Шаг 1. Регистрация на сайте

Перейдите на официальный сайт СДЭК и нажмите «Войти». В форме регистрации укажите:

  • 📧 Активный email (на него придёт письмо с подтверждением)
  • 📱 Номер телефона (обязательно российский)
  • 🔐 Придумайте надёжный пароль (минимум 8 символов)

Шаг 2. Создание первой отправки

После регистрации система автоматически сгенерирует для вас СДЭК ID, но он станет активным только после первой отправки. Для этого:

  1. В личном кабинете выберите «Создать отправление»
  2. Заполните данные получателя и параметры посылки
  3. Оплатите заказ любым удобным способом

Шаг 3. Поиск идентификатора

Ваш СДЭК ID появится в разделе «Мои данные» → «Профиль». Он будет выглядеть как комбинация из 8–10 цифр, например: 12345678. Скопируйте его и сохраните в надёжном месте.

💡

Если вы не видите СДЭК ID в профиле, обновите страницу или очистите кэш браузера. Иногда система выдаёт идентификатор с задержкой до 1 часа после первой отправки.

Как получить СДЭК ID для бизнеса: инструкция для ИП и ООО

Для юридических лиц и индивидуальных предпринимателей процесс сложнее, но открывает доступ к расширенным возможностям. Главное отличие — обязательная верификация через менеджера СДЭК.

Шаг 1. Заключение договора

Без действующего договора с СДЭК бизнес-аккаунт создать невозможно. Вам потребуется:

  • 📄 Заполнить заявку на сайте в разделе «Для бизнеса»
  • 📋 Предоставить сканы ИНН, ОГРН/ОГРИП и выписки из ЕГРЮЛ/ЕГРИП
  • ✍️ Подписать договор (можно электронной подписью)

Шаг 2. Активация личного кабинета

После подписания договора на ваш email придёт письмо с инструкцией по активации аккаунта. В нём будет временный пароль и ссылка для входа. При первом входе система запросит:

  • 🔄 Смену пароля на постоянный
  • 📞 Подтверждение телефона через SMS
  • 🏢 Указание юридического адреса компании

Шаг 3. Получение корпоративного СДЭК ID

Ваш идентификатор будет доступен в разделе «Настройки» → «Интеграции». Он отличается от личного ID формат: обычно начинается с префикса B- (например, B-987654321). Этот префикс указывает на бизнес-аккаунт и даёт доступ к API.

Что делать, если менеджер не отвечает?

Если вашу заявку на договор игнорируют дольше 3 рабочих дней, напишите в службу поддержки на email partner@cdek.ru с темой "Запрос на ускорение верификации". Приложите сканы документов и укажите номер заявки.

⚠️ Внимание: С 2026 года СДЭК блокирует бизнес-аккаунты, если в течение 30 дней после регистрации не было ни одной отправки. Чтобы избежать блокировки, создайте тестовое отправление (даже если оно не будет оплачено).

Где найти свой СДЭК ID: все способы

Идентификатор может понадобиться в самый неожиданный момент — например, при настройке интеграции с Wildberries или Ozon. Разберём, где его искать в зависимости от типа аккаунта.

1. Личный кабинет на сайте

Самый надёжный способ:

  1. Авторизуйтесь на cdek.ru
  2. Перейдите в «Мои данные» → «Профиль»
  3. Найдите строку «Ваш СДЭК ID» (или «Идентификатор клиента»)

2. Мобильное приложение

В официальном приложении СДЭК (доступно для iOS и Android):

  • 📱 Откройте боковое меню (свайп справа)
  • 👤 Выберите «Профиль»
  • 🔍 Прокрутите до блока «Данные аккаунта»

3. Письмо от СДЭК

При регистрации бизнес-аккаунта идентификатор приходит на email в письме с темой «Ваш корпоративный доступ активирован». Если письмо потеряно, проверьте папку «Спам» или запросите повторную отправку через поддержку.

4. Через API (для разработчиков)

Если вы уже подключены к API СДЭК, получить ID можно запросом:

GET https://api.cdek.ru/v2/oauth/token

Headers:


Authorization: Basic {ваш_client_id:ваш_client_secret}

В ответе будет поле "sub" — это и есть ваш СДЭК ID.

💡

Сохраните свой СДЭК ID в менеджере паролей (например, 1Password или KeePass). При утере восстановить его можно только через службу поддержки, что занимает до 5 рабочих дней.

Типичные ошибки при получении СДЭК ID и как их избежать

Даже в простом процессе регистрации клиенты часто сталкиваются с проблемами. Мы собрали самые распространённые ошибки и способы их решения.

Ошибка 1: «Идентификатор не найден» в личном кабинете

Причины:

  • 🕒 Вы ещё не создали первую отправку (для физлиц)
  • 📄 Договор не прошёл верификацию (для бизнеса)
  • 🔄 Система не успела сгенерировать ID (подождите 1–2 часа)

Решение: Проверьте статус аккаунта в разделе «Уведомления». Если там есть сообщение «Требуется подтверждение документов», свяжитесь с менеджером.

Ошибка 2: «Неверный формат СДЭК ID» при интеграции

Эта ошибка возникает, если вы пытаетесь использовать личный идентификатор для бизнес-функций. Например, при подключении к API маркетплейса. Проверьте:

  • 🔢 Длина ID (для бизнеса — 10 символов с префиксом B-)
  • 🔤 Отсутствие пробелов или тире (кроме префикса)

Ошибка 3: Блокировка аккаунта после регистрации

СДЭК автоматически блокирует подозрительные аккаунты. Частые причины:

  • 📍 Указание несуществующего адреса регистрации
  • 📞 Использование виртуального номера телефона
  • 🔄 Многократные попытки ввода неверного пароля

Решение: Напишите в поддержку с темой «Разблокировка аккаунта» и приложите скан паспорта.

Как ускорить верификацию бизнес-аккаунта?

Если вам нужен СДЭК ID срочно (например, для запуска рекламной кампании), укажите это в комментарии к заявке. Менеджеры приоритизируют такие запросы, сокращая срок проверки с 5 до 1–2 дней.

Как использовать СДЭК ID: практические примеры

Получить идентификатор — половина дела. Рассмотрим, где и как его применять для максимальной эффективности.

1. Отслеживание посылок в реальном времени

С СДЭК ID вы можете настроить автоматическое отслеживание всех ваших отправлений через:

  • 📊 Google Таблицы (с помощью скрипта Apps Script)
  • 📱 Телеграм-боты (например, @CDEK_Tracker_Bot)
  • 🖥️ CRM-системы (Битрикс24, amoCRM)

2. Интеграция с маркетплейсами

Для продавцов на Ozon, Wildberries или Яндекс Маркет СДЭК ID нужен для:

  • 📦 Автоматической передачи данных о заказах в СДЭК
  • 💰 Получения скидок на доставку (до 30% для партнёров)
  • 📈 Синхронизации статусов отправлений с личным кабинетом маркетплейса

3. Работа с API СДЭК

Для разработчиков идентификатор является обязательным параметром в большинстве запросов. Пример авторизации:

POST https://api.cdek.ru/v2/oauth/token

Headers:


Content-Type: application/x-www-form-urlencoded


Body:


grant_type=client_credentials


client_id=ваш_СДЭК_ID


client_secret=ваш_секретный_ключ

⚠️ Внимание: Никогда не передавайте свой СДЭК ID и client_secret третьим лицам. При компрометации данных злоумощник сможет создавать отправления от вашего имени, что приведёт к финансовым потерям.

4. Печать наклеек и документов

С идентификатором вы можете:

  • 🖨️ Печатать наклейки пачками (до 100 шт. за раз)
  • 📄 Генерировать акты приёма-передачи в PDF
  • 📋 Создавать сводные ведомости для бухгалтерии
💡

Для удобства добавьте свой СДЭК ID в шаблоны документов Word или Excel. Это сэкономит время при заполнении форм для массовых отправок.

FAQ: Частые вопросы о СДЭК ID

Можно ли использовать один СДЭК ID для нескольких магазинов?

Нет, каждый юридический субъект (ИП или ООО) должен иметь отдельный идентификатор. Если у вас несколько магазинов на одном ИП, допускается использование одного СДЭК ID, но это усложнит аналитику по отправкам. Оптимальное решение — создать подаккаунты в личном кабинете.

Что делать, если потерял СДЭК ID?

Для физических лиц: идентификатор можно найти в профиле или запросить повторно через поддержку (ответ в течение 24 часов). Для бизнес-аккаунтов: свяжитесь с вашим персональным менеджером в СДЭК — он предоставит данные после подтверждения личности.

Меняется ли СДЭК ID при смене тарифного плана?

Нет, идентификатор присваивается один раз и остаётся неизменным даже при смене тарифа или типа аккаунта. Исключение — полная смена юридического лица (например, реорганизация ООО в ИП), в этом случае потребуется новый договор и, соответственно, новый СДЭК ID.

Можно ли передать свой СДЭК ID другому человеку?

Передача идентификатора третьим лицам категорически запрещена правилами СДЭК. При обнаружении совместного использования аккаунта (например, если с одного ID создаются отправления из разных городов) компания вправе заблокировать доступ без предупреждения. Для совместной работы оформите дополнительные подаккаунты для сотрудников.

Сколько стоит получение СДЭК ID?

Для физических лиц — бесплатно. Для юридических лиц и ИП требуется заключение договора, стоимость которого зависит от тарифного плана. Минимальная цена — 0 рублей (при условии ежемесячного оборота от 50 отправлений). Подробные тарифы смотрите на странице «Условия для бизнеса».
