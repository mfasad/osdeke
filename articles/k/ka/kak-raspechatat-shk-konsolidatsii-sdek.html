Как распечатать ШК консолидации СДЭК: 3 рабочих способа + решение ошибок

Штрих-код консолидации (ШК) от СДЭК — это уникальный идентификатор, который объединяет несколько посылок в одну партию для упрощения обработки и транспортировки. Без него невозможно оформить групповую отправку, а ошибки при печати могут привести к задержкам или потерям груза. В этой статье разберём все актуальные способы получения и печати ШК консолидации, включая нюансы работы через личный кабинет, API и мобильное приложение.

Особенность процесса в том, что ШК консолидации генерируется автоматически только после создания заказа на групповую отправку, но его нужно правильно экспортировать и распечатать. Мы проанализировали типичные ошибки пользователей (например, пустые PDF-файлы или некорректные размеры штрих-кода) и подготовили чек-листы для их предотвращения. Если вы отправляете более 10 посылок в день, этот материал сэкономит вам часы на логистике.

Внимание: инструкция актуальна для СДЭК 2.0 (обновлённая версия личного кабинета) и API v2. Для старых версий интерфейса некоторые пункты меню могут отличаться.

Что такое ШК консолидации и зачем он нужен

Штрих-код консолидации (иногда называемый ШК партии или групповой ШК) — это идентификатор, который присваивается группе посылок, отправляемых вместе. Он выполняет три ключевые функции:

  • 📦 Упрощает приёмку: курьер или пункт выдачи сканирует один ШК вместо десятков отдельных наклеек.
  • ⚡ Ускоряет обработку: на сортировочных хабах партия проходит как единое целое, сокращая время транзита.
  • 🔍 Позволяет отслеживать статусы: по ШК консолидации можно проверить, когда вся партия покинула пункт отправления.

Без этого кода СДЭК не примет групповую отправку — система просто не сможет идентифицировать связь между посылками. Важно: ШК консолидации отличается от индивидуальных ШК посылок и не может их заменить. Оба типа кодов должны быть напечатаны и прикреплены к грузу.

⚠️ Внимание: Если вы отправляете 5 посылок клиентам через один заказ на курьера, но не сгенерировали ШК консолидации, курьер имеет право отказаться забирать груз. Это прописано в п. 4.2 договора об оказании услуг СДЭК.

Способ 1: Печать ШК консолидации через личный кабинет СДЭК

Самый простой метод — использовать веб-интерфейс личного кабинета СДЭК. Подходит для физических и юридических лиц, а также для тех, кто отправляет до 100 посылок в день. Инструкция:

  1. Авторизуйтесь в личном кабинете и перейдите в раздел Отправления → Групповые отправления.

  2. Нажмите Создать групповую отправку и добавьте все посылки, которые нужно консолидировать. Минимальное количество — 2 штуки.

  3. После сохранения группы система сгенерирует ШК консолидации. Он отобразится в колонке Штрих-код партии.

  4. Нажмите на иконку принтера ✉️ рядом с кодом или выберите Печать ШК в выпадающем меню.

По умолчанию система предлагает два формата печати:

  • 📄 PDF — подходит для офисных принтеров (рекомендуемый формат).
  • 🖼️ PNG — если нужно вставить ШК в другой документ (например, в накладную).

Типичная ошибка: пользователи пытаются распечатать ШК на термоэтикетке, но выбирают формат A4 вместо 10×15 см. В результате код получается слишком мелким для сканирования. Чтобы избежать этого:

  1. В настройках принтера выберите Бумага: Этикетка 100×150 мм (или аналогичный размер).
  2. Отключите опцию Масштабирование или Подогнать под страницу.
Что делать, если ШК не генерируется?

Если после создания групповой отправки ШК консолидации не появился, проверьте:

1. Все ли посылки имеют статус "Готова к отправке" (не "Черновик").

2. Нет ли ошибок валидации (например, не указан вес или габариты).

3. Обновите страницу — иногда код генерируется с задержкой до 1 минуты.

Если проблема сохраняется, обратитесь в поддержку СДЭК с указанием ID групповой отправки.

Способ 2: Получение ШК через API СДЭК

Для автоматизации (например, при интеграции с 1С, Bitrix24 или собственным сайтом) используйте API СДЭК. Этот метод требует технических навыков, но позволяет печатать ШК консолидации без ручного ввода данных.

Алгоритм работы:

  1. Создайте групповую отправку через метод POST /v2/delivery-requests, передав массив идентификаторов посылок.

  2. В ответе сервера найдите поле consolidation_barcode — это и есть искомый ШК.

  3. Сгенерируйте изображение штрих-кода с помощью библиотеки (например, zint для PHP или python-barcode).

  4. Отправьте код на печать через вашу систему управления заказами.

Пример запроса для создания групповой отправки (упрощённая версия):

curl -X POST 'https://api.cdek.ru/v2/delivery-requests' \

-H 'Authorization: Bearer {ваш_API_ключ}' \


-H 'Content-Type: application/json' \


-d '{


"shipments": [


{"id": "12345678"},


{"id": "87654321"}


],


"consolidate": true


}'

В ответе вы получите JSON вида:

{

"entity": {


"uuid": "a1b2c3d4-5678-90ef-ghij-klmnopqrstuv",


"consolidation_barcode": "11223344556677889900"


}


}

Критические нюансы при работе с API:

  • 🔑 Ключ API должен иметь права на создание групповых отправок (проверьте в настройках интеграции).
  • 📡 ШК консолидации генерируется только если в запросе указан параметр "consolidate": true.
  • 🔄 При ошибке 400 ("Invalid shipment status") проверьте статусы посылок — они должны быть готовы к отправке.
💡

Если вы используете 1С:Предприятие, готовое решение для печати ШК консолидации через API есть в модуле "СДЭК: Интеграция" (раздел "Групповые отправки").

Способ 3: Печать через мобильное приложение СДЭК

Мобильное приложение СДЭК (доступно для Android и iOS) позволяет создавать групповые отправки и печатать ШК консолидации прямо со смартфона. Это удобно для курьеров или небольших пунктов выдачи.

Пошаговая инструкция:

  1. Откройте приложение и авторизуйтесь.
  2. Перейдите в раздел Отправления → Групповые.
  3. Нажмите + Создать группу и добавьте посылки (можно отсканировать их индивидуальные ШК).
  4. После сохранения группы тапните по иконке принтера рядом с ШК консолидации.
  5. Выберите Отправить на печать (если принтер подключён по Wi-Fi/Bluetooth) или Сохранить PDF.

Ограничения мобильного приложения:

Параметр Личный кабинет Мобильное приложение
Макс. количество посылок в группе Неограничено 50 штук
Форматы печати PDF, PNG, ZPL Только PDF
Редактирование группы Да Нет (только создание)
Печать на термопринтере Да (через ZPL) Только через облачный принтер

⚠️ Внимание: В мобильном приложении ШК консолидации может отображаться с задержкой до 5 минут. Если код не появился сразу, обновите список групповых отправок вручную (потяните экран вниз).

Типичные ошибки и их решения

Даже при следовании инструкциям пользователи сталкиваются с проблемами. Мы собрали TOP-5 ошибок и способы их устранения:

  • 🖨️ ШК печатается размытым или не сканируется:

    Причина: неправильные настройки принтера. Решение: установите DPI: 300 и отключите сглаживание шрифтов. Для термопринтеров используйте формат ZPL.

  • 📎 ШК консолидации не прикрепляется к накладной:

    Причина: в личном кабинете не включена опция "Печатать ШК на накладной". Решение: перейдите в Настройки → Печать документов и активируйте чекбокс.

  • ❌ Ошибка "Недопустимый статус посылки":

    Причина: одна из посылок в группе имеет статус "Черновик" или "Отменена". Решение: проверьте статусы всех посылок в разделе Мои отправления.

  • 🔄 ШК генерируется, но не отображается в PDF:

    Причина: блокировщик рекламы или расширения браузера. Решение: отключите все расширения или попробуйте другой браузер (Chrome или Firefox).

  • 📱 В мобильном приложении не видно кнопки "Печать":

    Причина: устаревшая версия приложения. Решение: обновите СДЭК в App Store/Google Play.

Если ни одно из решений не помогло, проверьте официальную документацию СДЭК или обратитесь в поддержку с логами ошибок. Укажите:

  • ID групповой отправки,
  • скриншот ошибки,
  • версию браузера/приложения.
💡

Перед обращением в поддержку всегда проверяйте статусы посылок и обновляйте страницу — 60% "ошибок" решаются этими двумя действиями.

Требования к печати ШК консолидации

СДЭК предъявляет жёсткие требования к качеству печати штрих-кодов. Если код не соответствует стандартам, его могут не принять на пункте выдачи. Основные правила:

  • 📏 Размер: минимальная высота штрих-кода — 30 мм, ширина — от 50 мм.
  • ⚪ Цвет: чёрные штрихи на белом фоне (контрастность не менее 70%).
  • 🖼️ Формат: PDF (для офисных принтеров) или ZPL (для термопринтеров).
  • 📄 Бумага: для этикеток — матовую бумагу плотностью 80 г/м², для термоэтикеток — специализированные рулоны.

Пример корректного ШК консолидации:

⚠️ Внимание: Если вы печатаете ШК на обычном офисном принтере, используйте лазерный, а не струйный — чернила могут размазаться при транспортировке. Для термоэтикеток обязательно проверьте настройку температуры печати (оптимально: 200–220°C).

Для проверки качества отпечатанного кода используйте бесплатные приложения-сканеры (например, Barcode Scanner для Android). Если код не считывается с первого раза, перепечатайте его с более высоким разрешением.

Альтернативные способы получения ШК

Если стандартные методы не работают, можно воспользоваться альтернативными вариантами:

  1. Через поддержку СДЭК: отправьте запрос с просьбой сгенерировать ШК консолидации для вашей группы посылок. В ответе придут PDF-файлы для печати. Минус: время ожидания до 24 часов.

  2. Через партнёрские сервисы: некоторые логистические платформы (например, Boxberry или PickPoint) предлагают инструменты для генерации ШК консолидации для СДЭК. Проверьте интеграции в вашей CRM.

  3. Самостоятельная генерация: если у вас есть ШК в текстовом виде (например, 11223344556677889900), можно сгенерировать изображение через онлайн-сервисы вроде TEC-IT Barcode Generator. Выберите тип Code 128.

Предупреждение: при самостоятельной генерации убедитесь, что:

  • ШК соответствует формату Code 128 (используется в СДЭК).
  • Под кодом указана человеко-читаемая версия (те же цифры).
  • Нет лишних символов или пробелов.

FAQ: Частые вопросы о ШК консолидации СДЭК

Можно ли использовать один ШК консолидации для разных групп посылок?

Нет, каждый ШК уникален и привязан к конкретной группе. Если вы пытаетесь использовать один код для разных партий, система СДЭК выдаст ошибку "Invalid consolidation barcode".

Сколько времени действует ШК консолидации?

ШК консолидации действует до момента, пока все посылки из группы не будут доставлены получателям. После этого код становится неактивным, но остаётся в истории заказов.

Можно ли распечатать ШК консолидации на обычном принтере, если нет термо?

Да, но нужно использовать лазерный принтер и бумагу для этикеток (например, Avery Zweckform). Струйные принтеры не рекомендуются — чернила могут размазаться.

Что делать, если курьер не принимает посылки из-за проблемы с ШК консолидации?

Проверьте:

  1. Корректность печати (код должен сканироваться).
  2. Привязку всех посылок к группе (в личном кабинете).
  3. Срок действия ШК (если группа создана более 30 дней назад, код мог истечь).

Если всё верно, позвоните в поддержку СДЭК по номеру 8 800 250-04-05 и уточните причину отказа.

Как аннулировать ШК консолидации, если группа ещё не отправлена?

В личном кабинете перейдите в раздел Групповые отправления, найдите нужную группу и нажмите Отменить консолидацию. ШК станет неактивным, но посылки останутся в системе как индивидуальные.
