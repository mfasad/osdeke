Интеграция Авито с СДЭК: как подключить и настроить доставку за 15 минут

Продажи на Aviito растут, но ручное оформление заказов через СДЭК отнимает часы каждый день? Автоматизация доставки решает эту проблему: заказы из Aviito моментально попадают в личный кабинет СДЭК, а клиенты получают трек-номера без вашего участия. Но как настроить эту связку правильно, чтобы избежать ошибок с тарифами, адресами или статусами?

Эта статья — пошаговая инструкция с актуальными скриншотами 2026 года, которая поможет подключить Aviito к СДЭК через API или готовые модули (например, МойСклад или Битрикс24). Мы разберём:

  • 🔹 3 способа интеграции (ручная, полуавтоматическая, полная автоматизация)
  • 🔹 Пошаговую настройку API с примерами кода для разработчиков
  • 🔹 Типичные ошибки при подключении и как их исправить (коды 403, 500, проблемы с токенами)
  • 🔹 Сравнение тарифов СДЭК для заказов с Aviito (включая скидки для бизнес-аккаунтов)

Важно: если вы используете Aviito Доставку, часть функций уже интегрирована с СДЭК по умолчанию — но кастомизация маршрутов, упаковки или оплаты требует отдельной настройки. Начнём с выбора способа подключения.

1. Способы подключения Авито к СДЭК: какой выбрать?

Интеграция Aviito и СДЭК возможна тремя путями. Выбор зависит от объёма заказов, технических навыков и бюджета:

Способ Сложность Стоимость Подходит для
Ручной ввод (копирование заказов в ЛК СДЭК) ⭐ Бесплатно 1–10 заказов/день
Готовые модули (МойСклад, Битрикс24, RetailCRM) ⭐⭐ От 500 ₽/мес 10–100 заказов/день
Прямое API (самостоятельная разработка) ⭐⭐⭐ От 10 000 ₽ (разово) 100+ заказов/день

🔸 Ручной ввод подойдёт новичкам: вы экспортируете заказы из Aviito в Excel и загружаете их в СДЭК через файл. Минус — высокая вероятность ошибок при копировании адресов или веса.

🔸 Готовые модули (например, МойСклад) синхронизируют заказы автоматически, но требуют ежемесячной оплаты. Подходит для среднего бизнеса.

🔸 Прямое API — самый гибкий вариант для крупных продавцов. Позволяет настраивать логику упаковки, выбор тарифов и уведомления клиентов. Требует знаний программирования или помощи разработчика.

2. Пошаговая настройка API СДЭК для Авито

Если вы выбрали прямую интеграцию через API, следуйте этому алгоритму. Для работы понадобится:

  • 🔐 Ключ API СДЭК (получается в личном кабинете)
  • 📝 Данные для аутентификации (client_id и client_secret)
  • 💻 Доступ к серверу (для размещения скрипта синхронизации)

📌 Шаг 1. Регистрация приложения в СДЭК

  1. Перейдите в Личный кабинет СДЭК → API → Мои приложения.
  2. Нажмите «Создать приложение» и укажите:
    • 📌 Название (например, «Интеграция с Авито»)
    • 📌 Redirect URI — https://your-domain.ru/cdek/callback (замените на свой домен)
    • 📌 Права доступа: orders:read, orders:write, delivery_points:read
  • Скопируйте client_id и client_secret — они понадобятся для авторизации.

    • 📌 Шаг 2. Получение токена доступа

    Отправьте POST-запрос на эндпоинт авторизации:

    curl -X POST 'https://api.cdek.ru/v2/oauth/token' \

    -H 'Content-Type: application/x-www-form-urlencoded' \
    

    -d 'grant_type=client_credentials&client_id=ВАШ_CLIENT_ID&client_secret=ВАШ_CLIENT_SECRET'

    В ответе вы получите access_token (действует 1 час) и refresh_token (для продления сессии). Сохраните их в безопасном месте.

    📌 Шаг 3. Настройка вебхуков в Авито

    Чтобы заказы автоматически передавались в СДЭК, настройте уведомления в Aviito:

    1. Перейдите в Личный кабинет Авито → Настройки → API.
    2. Добавьте вебхук с URL вашего сервера (например, https://your-domain.ru/api/avito-order).
    3. Укажите события для отслеживания:
      • 📦 order.created — новый заказ
      • 📦 order.status_changed — изменение статуса
    Пример кода для обработки вебхука Авито

    // Пример обработчика вебхука на PHP

    3. Настройка тарифов и услуг СДЭК для заказов с Авито

    Ошибка в выборе тарифа — одна из главных причин переплат или проблем с доставкой. В СДЭК для заказов с Aviito доступны специальные условия:

    Тариф Срок доставки Стоимость (от) Особенности
    Экономичный 3–7 дней 250 ₽ Для товаров весом до 10 кг
    Стандартный 2–5 дней 350 ₽ Страховка до 50 000 ₽ включена
    Экспресс 1–2 дня 600 ₽ Доставка в день заказа (до 12:00)
    Посылка (почтовая) 5–14 дней 180 ₽ Для удалённых регионов

    ⚠️ Внимание: Если вес или габариты товара в заказе с Aviito не совпадают с данными в СДЭК, система автоматически пересчитает стоимость доставки. Чтобы избежать доплат:

    • 📏 Указывайте точный вес (с упаковкой!) в карточке товара на Aviito.
    • 📦 Для хрупких товаров выбирайте тариф со страховкой (например, «Стандартный»).
    • 🚚 Если отправляете крупногабарит (длина > 1.5 м), уточните возможность доставки в калькуляторе СДЭК.
    💡

    Используйте шаблоны упаковки в ЛК СДЭК, чтобы не вводить габариты каждого заказа вручную. Например, создайте шаблон «Коробка 30×20×10 см» для мелких товаров и привяжите его к тарифу «Экономичный».

    4. Типичные ошибки при подключении и их решения

    Даже при правильной настройке API могут возникать сбои. Вот 5 самых частых ошибок и способы их исправления:

    ⚠️ Внимание: Если при отправке заказа в СДЭК вы получаете ошибку 403 Forbidden, проверьте scope вашего токена. Часто проблема в недостаточных правах (нужны orders:write).
    Ошибка Причина Решение
    500 Internal Server Error Некорректный формат JSON или отсутствует обязательное поле (например, recipient) Проверьте структуру запроса в документации СДЭК
    400 Bad Request: "tariff_id not found" Указан несуществующий тариф или он недоступен для выбранного направления Используйте /v2/delivery/tariff-list для получения актуальных тарифов
    401 Unauthorized Истёк access_token или неверные client_id/client_secret Обновите токен (см. Шаг 2) или проверьте ключи в ЛК СДЭК
    Заказ не отображается в ЛК СДЭК Вебхук Aviito не сработал или данные не прошли валидацию Проверьте логи сервера и повторно отправьте запрос с debug: true

    5. Автоматизация уведомлений клиентов

    После создания заказа в СДЭК клиент должен получить:

    1. 📧 Email с номером заказа и трек-номером.
    2. 📱 SMS с ссылкой на отслеживание.
    3. 🔔 Уведомление в Авито (через чат).

    📌 Способ 1: Настройка в ЛК СДЭК

    Перейдите в Личный кабинет → Настройки → Уведомления и включите опции:

    • 🔘 «Отправлять SMS при создании заказа»
    • 🔘 «Отправлять email с трек-номером»
    • 🔘 «Интеграция с Авито» (если подключен модуль)

    📌 Способ 2: Кастомизация через API

    Добавьте в запрос создания заказа поле notification:

    "notification": {

    "email": "client@example.com",
    

    "phone": "+79991234567",
    

    "send_sms": true,
    

    "send_email": true
    

    }

    1) Корректность email/телефона в заказе.

    2) Баланс на счёте СДЭК (SMS платные — ~1.5 ₽ за штуку).

    3) Настройки спам-фильтров (особенно для email с домена @cdek.ru).-->

    6. Оптимизация стоимости доставки: скидки и бонусы

    СДЭК предлагает скидки для продавцов с Авито при выполнении условий:

    • 🎯 Объём от 50 заказов/мес — скидка 5% на тарифы.
    • 🎯 Объём от 200 заказов/мес — скидка 10% + бесплатная упаковка.
    • 🎯 Оплата по безналу — дополнительные 2% кэшбэка.

    📊 Как получить скидку?

    1. Зарегистрируйте бизнес-аккаунт в СДЭК (через форму на сайте).
    2. Привяжите свой аккаунт Aviito в разделе «Интеграции».
    3. Отправьте не менее 50 заказов за месяц — скидка активируется автоматически.

    ⚠️ Внимание: Скидки не суммируются с акциями (например, «Бесплатная доставка по России»). Если у вас действует промокод от Aviito, уточните приоритет скидок у менеджера СДЭК.

    💡

    Используйте консолидацию заказов — отправляйте несколько товаров одному клиенту в одной коробке. Это снизит стоимость доставки на 15–30%.

    Битрикс24 От 1 990 ₽/мес CRM + интеграция с Авито и СДЭК, аналитика продаж

    • 🛒 Для маленького магазина (до 50 заказов/день) хватит МойСклад.

    • 🏭 Для крупных продавцов (200+ заказов) лучше RetailCRM или собственное API.

    FAQ: Частые вопросы по интеграции Авито и СДЭК

    ❓ Как проверить, что интеграция работает?

    Создайте тестовый заказ на Aviito (например, через «Купить в 1 клик»). Если он появился в ЛК СДЭК в течение 5 минут — интеграция настроена верно. Также проверьте логи вашего сервера на ошибки.

    ❓ Можно ли подключить СДЭК к Авито без API?

    Да, через сервисы-посредники (МойСклад, Битрикс24) или ручной экспорт заказов в Excel. Однако без API вы не сможете:

    • 🔹 Автоматически обновлять статусы заказов.
    • 🔹 Настраивать гибкие тарифы в зависимости от веса/региона.
    • 🔹 Отправлять кастомизированные уведомления клиентам.
    ❓ Почему СДЭК не принимает заказ из Авито?

    Частые причины:

    1. 📌 Неверный адрес (отсутствует индекс или город).
    2. 📌 Превышен лимит веса (максимум 31.5 кг для стандартных тарифов).
    3. 📌 Недостаточно средств на счёте СДЭК.
    4. 📌 Некорректный tariff_code (уточните актуальные коды в документации).

    Проверьте ответ API на ошибки — там будет подробное описание проблемы.

    ❓ Как отменить заказ в СДЭК, если клиент отказался?

    Отмена возможна, если заказ ещё не передан курьеру. Для этого:

    1. Перейдите в Личный кабинет СДЭК → Заказы.
    2. Найдите заказ и нажмите «Отменить».
    3. Укажите причину (например, «Отказ клиента»).

    Если заказ уже в пути, свяжитесь с поддержкой СДЭК по телефону 8 800 250-14-95.

    ❓ Сколько стоит интеграция Авито с СДЭК?

    Стоимость зависит от способа подключения:

    • 🆓 Ручной ввод — бесплатно.
    • 💰 Готовые модули — от 500 ₽/мес (например, МойСклад).
    • 💻 Разработка API — от 10 000 ₽ (разово) + возможные доработки.

    Дополнительно оплачиваются:

    • 📦 Стоимость доставки по тарифам СДЭК.
    • 📞 SMS-уведомления (~1.5 ₽ за штуку).
